# front

This is front-end of project [WeIns](https://github.com/DDIP2020)

### Files structure:

```text
components/
    blog            // 动态展示框
    footer          // 底边栏
    issue           // 动态展示页面
    dissue          // discover issue
    sign*Form       // 登录/注册的表单
    topnav          // 顶部导航栏
    release_blog    // 发布动态
    release_comment // 发表评论
    counter         // 计数器组件
    personalcard    // 个人主页的展板
    share           // 转发的弹出框
    comment         // 评论组件
    info            // 个人资料完整度
    information     // 个人资料
    myblogs         // 我的动态
    followers       // 粉丝列表
    followings      // 关注列表
    followcard      // 关注/粉丝的组件
    userTable       // 用户列表，供管理员查询
    other*          // 访问其它用户主页用到的组件

views/
    home            // 主页
    signup          // 注册界面
    personcenter    // 个人中心
    otherperson*    // 访问他人主页
    manager         // 用户管理中心
    discover        // 发现（分类浏览）
```
